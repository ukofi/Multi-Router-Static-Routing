PC1 is in Network A (e.g., 192.168.1.0/24)

R1 – R2 link is in Network B (e.g., 10.1.1.0/30)

R2 – R3 link is in Network C (e.g., 10.1.2.0/30)

PC2 is in Network D (e.g., 192.168.2.0/24)

Each router knows only its directly connected networks. Static routes tell them how to reach the remote networks.
